HTML - HyperText Markup Language

HTML stands for HyperText Markup Language. It is used to create web pages. There are many html elements which tell browser how to display the contents.

Here is a list of a few HTML elements:

  • html element is the root element.
  • head element contains meta information about the HTML page.
  • title element is used to give tite to the HTML page.
  • body element defines the contents to be displayed on the webpage.
  • footer element is for the footer information like copyrights etc.
